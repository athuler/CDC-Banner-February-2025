# CDC Banner (February 2025)
 
## Overview

In February 2025, one of the following banners appeared accross various pages on the CDC's website:

Version 1:
> CDC's website is being modified to comply with President Trump's Executive Orders.

Version 2:
> Per a court order, HHS is required to restore this website as of 11:59PM ET, February 11, 2025.

that there are two sexes, male and female. The Trump Administration rejects gender ideology and condemns the harms it causes to children,
